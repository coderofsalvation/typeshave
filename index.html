Typeshave

Guard your function's incoming data using typeshave wrappers in JAVASCRIPT & PHP.

Prevent functions from exploding with garbage-in garbage-out.

BROWSER

<script src="https://raw.githubusercontent.com/coderofsalvation/typeshave.js/master/browser/typeshave.min.js"></script>
<script>
  typeshave = require("typeshave").typesafe;

  var foo = typeshave({
    foo: { type: "string" },
    bar: { type: "boolean" }
  }, function(foo,bar){
    alert("ok data passed!");
  });

  foo( "string", true );
</script>
